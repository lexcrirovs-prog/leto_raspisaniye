# Летний планировщик 2026

Статический сайт для детского летнего расписания на июнь, июль и август 2026.

## Локальный запуск

```bash
python -m http.server 3100
```

После запуска открыть:

```text
http://localhost:3100/raspgpt/
```

## Проверка

```bash
npm test
```

Проверяются основные правила расписания:

- математика во вторник и четверг, 17:00-18:00;
- дорога к математике 16:30-17:00 и назад 18:00-18:30;
- гребля по понедельникам, средам и пятницам;
- курс и генеалогическое древо только в июне;
- чтение, еда, прогулка и телефон есть каждый день;
- воскресенье без учебных проектов, репетитора и гребли.

## Деплой на kotelgavno.ru

Сайт не требует Node.js на сервере. Для адреса `https://kotelgavno.ru/raspgpt/` достаточно скопировать папку `raspgpt` в веб-корень домена:

```text
/home/p/premiuig/kotelgavno.ru/public_html/raspgpt
```

Корневой `index.html` можно положить в `public_html`, если нужен редирект с главной страницы на `/raspgpt/`.
